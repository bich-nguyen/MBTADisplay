## Local Development
1. Create a ".env" file in the roject root and fill in {}:

MBTA_API_KEY = "{your_api_key}"

WEATHER_CONTACT = "{email@domain.com}"


3. Install dependencies & start the server

    npm install
   
    npm start


5. Open in brower:

http://localhost:3000
